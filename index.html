TheraPC

Welcome to TheraPC! TheraPC is a program that wants to make you feel better and solve your problems.

What is TheraPC?

What is it?

TheraPC is an individual software project that accepts your current emotion and personal problem to return a song to listen to and resources it finds helpful.

How does it do it?

TheraPC activates a program with a JavaFX GUI that prompts the user to input an emotion. It will check its' MongoDB mood database for reference, and will find an appropriate song to give back in its' MongoDB song database. It will then prompt the user to input an explanation to their problem, scan through for keywords, and search for resources on the internet to return.

Technical Features

  • Adorable eye-pleasing JavaFX graphics
  • MongoDB Java Driver
  • ElasticSearch API
  • Google CustomSearch API
